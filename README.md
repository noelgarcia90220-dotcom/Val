# Ojos Bonitos by Valerie 💗💜

Simple lash business website ready for GitHub Pages.

## Files
- index.html → main website
- lash.jpg → your lash image (add your own)

## How to use on GitHub Pages
1. Create a new GitHub repository
2. Upload index.html and lash.jpg
3. Go to Settings → Pages
4. Select branch: main / root
5. Save and wait for link
